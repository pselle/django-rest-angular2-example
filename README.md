# Angular 2 and Django REST Framework

Set up the Django app:

- Create a virtualenv
- `pip install -r requirements.txt`
- `cd exampleapp`
- `python manage.py migrate`
- `python manage.py loaddata sample-dinosaurs.json`
- `python manage.py runserver`

Set up Angular app:

- `cd angular2`
- `npm install`
- `npm start`
